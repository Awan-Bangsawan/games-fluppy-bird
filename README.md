# Flappy Bird Game

## Deskripsi

Flappy Bird adalah game sederhana yang menguji refleks dan ketepatan pemain. Pemain harus mengendalikan seekor burung agar dapat melewati celah di antara pipa-pipa tanpa menabraknya. Setiap pipa yang berhasil dilewati akan menambah skor pemain.

## Fitur

* Kontrol burung dengan satu tombol.
* Sistem skor otomatis.
* Tampilan game yang sederhana dan mudah dimainkan.
* Deteksi tabrakan dengan pipa dan batas layar.
* Game Over ketika burung menabrak rintangan.

## Teknologi yang Digunakan

* HTML
* CSS
* JavaScript

## Cara Menjalankan Project

1. Download atau clone repository ini.
2. Buka folder project.
3. Jalankan file `index.html` menggunakan browser.
4. Game siap dimainkan.

## Cara Bermain

* Tekan **Spasi (Space)** atau klik mouse untuk membuat burung terbang ke atas.
* Hindari tabrakan dengan pipa.
* Kumpulkan skor sebanyak mungkin dengan melewati setiap pipa.
